Nursery Management System

Languages, Database, Framework Used
Info
The Nursery Management System is designed to streamline administrative and educational tasks in a kindergarten setting. It allows parents to register their children, track attendance, and manage fee payment. Teachers can manage student records, update attendance, and communicate with parents, while administrators oversee all operations, including student enrollment, staff management, and report generation. PHP handles server-side logic for secure data processing, JavaScript provides interactivity and real-time updates, HTML and CSS ensure a responsive and visually appealing interface, and MySQL manages the database, storing student records, attendance logs, and communication histories, showcasing expertise in full-stack web development and efficient database management.

Clinic Appointment System

Languages, Database, Framework Used
Info
The Clinic Appointment System is a web application designed using Java for the backend, PostgreSQL as the database, and Bootstrap for the frontend. It allows patients to register, log in, and book appointments with doctors based on available time slots. Doctors can manage their schedules, view appointments, and update their availability. The system features role-based access control, separate dashboards for patients and doctors, and email notifications for appointment confirmations and reminders, providing a user-friendly and efficient way to manage clinic appointments.
